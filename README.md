# Blockchain Academy

제2서울핀테크랩에서 진행하는 **Blockchain Academy** 교육과정의 학습 내용과 실습 코드를 정리하는 저장소입니다.

블록체인의 기본 개념부터 이더리움 스마트 컨트랙트, Web3 프로그래밍, NFT까지 이론과 실습을 통해 학습합니다.

> 교육기간: 2026년 7월 20일 ~ 7월 22일  
> 본 저장소는 개인 학습 및 복습을 위한 비공식 기록입니다.

## Curriculum

### Day 1 — 블록체인 기초와 이더리움

#### 블록체인 기본 개념과 트렌드

- What and Why Blockchain?
- 블록체인과 금융
- STO와 RWA
- Web3와 AI의 융합

#### 이더리움 튜토리얼

- Remix IDE 사용법
- Solidity 스마트 컨트랙트 작성
- 데이터 저장 및 조회
- 이더리움 및 토큰 전송
- 블록체인 투표 기능 구현

### Day 2 — 블록체인 기술과 Web3 프로그래밍

#### 블록체인 핵심 기술

- 비트코인 코어 살펴보기
- 블록체인의 주요 보안 기술
- 스마트 컨트랙트와 이더리움

#### Web3 활용 프로그래밍

- 이더리움 기반 NFT 게임
- Solidity 스마트 컨트랙트
- Remix IDE
- MetaMask
- VS Code 및 Node.js

### Day 3 — STO, CBDC와 NFT

#### 블록체인 주요 활용 분야

- STO(Security Token Offering)
- CBDC(Central Bank Digital Currency)
- NFT와 메타버스
- 국내 블록체인 주요 이슈

#### Web3 활용 프로그래밍 확장

- NFT 표준 라이브러리
- ERC-721 토큰
- OpenZeppelin 활용
- NFT 스마트 컨트랙트 작성 및 배포
